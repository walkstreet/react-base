# React 项目

> 基于React的框架

## 项目说明

+ 融入了react-router开发SPA，支持懒加载
+ redux管理数据流
+ Webpack则加入了图片转dataurl处理及sass/scss
+ 支持ES6及ES6+，eslint校验
+ 支持热加载

## 构建命令

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:3000
npm run dev

# build for production with minification
npm run build
```
